# 편집 기법 팔레트 (다양성 메뉴 — Writer / Director / Cameraman 공용)

> 이건 **고를 수 있는 메뉴**다. 매 에피소드 의도에 맞는 기법을 **골라서** 써라.
> 한 가지(특히 원테이크/vlog)로 습관적으로 회귀하지 말 것 — **다양성이 목표**다.
> 각 기법은 `edit_effect` / `editing_concept` 신호로 Cameraman에 전달돼 실제 렌더가
> 달라진다. **기법이 의도를 정하지 않는다 — 의도가 기법을 고른다.**
> (판단 원칙은 editing_direction.md 참고.)

## 기법 메뉴

| 기법 | editing_concept / edit_effect | 어떤 의도에 맞나 | 언제 쓰나 |
|---|---|---|---|
| **롱테이크/원테이크** | `long_take` / `static` | 끊김 없는 한 순간의 공기·정서 | 단일 연속 모먼트가 그 자체로 이야기일 때. **기본값 아님, 소수(대략 1/3↓ 경향)** |
| **래피드 몽타주** | `rapid_montage` / `speed_1.3x` | 활기·리듬·여러 순간의 콜라주 | 짧은 순간 여러 개를 경쾌하게 엮을 때 |
| **배속 램프(×2~×3)** | `speed_2x` / `speed_3x` | 지루한 구간 압축해 setup→payoff를 한 컷에 | 긴 클립인데 결정적 순간이 늦게 올 때. **항상 빠르게가 아님 — 잔잔함이 의도면 금지** |
| **하루치 컴필레이션 vlog** | `themed_compilation` | "오늘 하루" 흐름, 일상 누적의 정 | 여러 클립을 시간/테마로 묶어 하루를 압축 (필요시 배속 병행) |
| **트위스트 엔딩** | `twist_ending` / `freeze_last_frame` | 예상 뒤집기·반전의 한 방 | 마지막에 의외의 결과가 있을 때, 끝 프레임을 freeze로 못박음 |
| **슬로모** | `slow_mo` | 결정적 동작의 감정 증폭 | 점프/달리기/장난의 하이라이트를 늘려 강조 |
| **비포-애프터** | `before_after` | 변화·대비(전/후) | 같은 대상의 두 상태를 나란히 보여줄 때 |
| **크로스 컷팅** | `cross_cutting` | 두 공간/주체를 교차로 긴장·대비 | 레오와 랴니가 다른 공간에서 동시에 뭔가 할 때 (공간전환은 캡션으로 narrate) |
| **스플릿 스크린** | `split_horizontal` / `split_vertical` | 두 장면 동시 비교 | secondary_asset_id로 두 컷을 한 화면에 |
| **켄번즈(팬/줌)** | `ken_burns` / `zoom` / `pan` | 정지 사진에 생명·시선 유도 | 사진(photo_i2v) 컷에 잔잔한 움직임 |

## 쓰는 법
1. 먼저 의도를 정한다(editing_direction.md §0).
2. 그 의도를 가장 잘 전달하는 기법을 위 메뉴에서 고른다 — **한 에피소드 안에서도
   컷마다 다른 edit_effect를 섞어** 단조로움을 피해라(도입 빠르게, 킥/엔딩은 여운).
3. 고른 기법은 의도를 섬기는지 자문하고, footage가 그 기법을 못 받쳐주면
   다른 기법으로 바꾸거나 Writer/Director 의도로 되돌아가 조율한다(루프).
4. 어떤 기법이든 **캡션 읽을 시간**(editing_direction.md §4)을 깨지 말 것.
